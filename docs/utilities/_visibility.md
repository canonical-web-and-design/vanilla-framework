---
collection: utilities
title: visibility
---

These utils exist to show or hide an element with a certain breakpoint, if specified. The modifier on each class relates directly to the breakpoint of the same name.

## .u-hidden--*

```css
.u-hidden
.u-hidden--small
.u-hidden--medium
.u-hidden--large
```

## .u-visible--*

```css
.u-visible
.u-visible--small
.u-visible--medium
.u-visible--large
```
